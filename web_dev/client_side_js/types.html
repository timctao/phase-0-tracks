Pokémon Types

There are eighteen different types of Pokémon, and each Pokémon can either be of just one type or dual-type. Each type has its own strengths and weaknesses, and should be memorized if you want to focus on battling in any Pokémon game. Click HERE if you'd like to see all the types of pokemon that can appear.

Of these types, each new generation of Pokémon trainers are allowed to pick a Pokémon from one of these three types; Fire, Water, or Grass. Here is a table detailing all the starting Pokémon you can get from each generation.

  Generation I Generation II Generation III Generation IV Generation V Generation VI
Grass Type Bulbasaur Chikorita Treecko Turtwig Snivy Chespin
Water Type Squirtle Totodile Mudkip Piplup Oshawott Froakie
Fire Type Charmander Cyndaquil Torchic Chimchar Tepig Fennekin

And of course, the most famous Pokémon of them all:

Pikachu!(Electric Type)
